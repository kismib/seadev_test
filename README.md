# Quyen Tran - React JS Test

## Timelines: 2 days

## How to Setup

1. `cd 2blocks`
2. Install the Application dependencies with `yarn install`

## How to run 2blocks on web browsers
1. `cd 2blocks`
2. `yarn start`
3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
